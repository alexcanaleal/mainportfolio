Atmos Station

Atmos Station is a simple 2D Galaga game made in C# Universal Windows with a roboust trading system and different planets to land at. Although this was the first large project I created in College with my peer, each year I update the game with better visuals and more optimized code. Currently, the game is in it's 3rd iteration.

Please note that Atmos is currently a UWP C# application. There is no way to play the game on the Web.

Code
Game Play

Atmos Station has a multitude of planets, and ships the player can use. From Atmos Station to Ignis Colony Ship, players are able to land at each station or planet and trade or upgrade resources and ships.

JSON

Saving user data is done with JSON. Account details such as Username are saved normally, but passwords are encoded. Ship details and player details are also saved in arrays.

Versions

So far, there have been 3 iterations of the game. The current version makes use of original vector images, and more dynamic gameplay.
