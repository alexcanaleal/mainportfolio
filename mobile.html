Java: Good Foods

Good Foods is a Restaurant search application. It makes use of the Zomato API to get all restaurants from a certain specified range. Users can then save their favorite restaurants. Information of each restaurant is displayed to the user in a dialog box, they can then either get directions or call the restaurant. The APP incorporates Room Database, Location Services, Call services, Email services, Navigation Drawer and Fragments.
