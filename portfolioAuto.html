Cypress

Where Selenium executes remote commands through the network, Cypress runs in the same run-loop as the application. Cypress tests anything that runs in a web browser. All of the architecture surrounding Cypress is built to handle modern JavaScript frameworks especially well. Using Cypress, I test all my Web Applications to make sure they provide a realiable user experiance.

I have used Cypress to test VIN Vault, Hungry 4 Restaurant, and a few smaller applications.
